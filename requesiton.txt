Requisição HTTP

URL: https://pokeapi.co/api/v2/pokemon?type=grass&name=i
    ${Endereço}/${path = caminho de identifição do recurso}

Request Method: GET | POST | PUT | delete

Query String
    metodo de busca e formatação para Requisição HTTP

?type=grass&name=i
    Requisição acima lista apenas pokemons tipo grama e que iniciem com a letra identifição

?offset=4&limit=5
    offset - inicio da Requisição
    limit - limite de Requisição apresentada

Request Header
    Configurações da API
    accept-language:
    pt-BR
    pt;q=0.9,
    en-US;q=0.8,
    en;q=0.7

Respense Header
    Configurações
